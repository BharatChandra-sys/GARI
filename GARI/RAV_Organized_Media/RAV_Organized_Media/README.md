# R.A.V Project Media Archive

This archive organizes the provided photos, videos, and reference screenshots into folders based on the project descriptions shown in the directions. Each folder contains related media and its own `README.md` file describing the contents.

## How Markdown Handles Media

- Photos can be displayed directly in Markdown using `![description](./file.jpg)`.
- Videos are linked in Markdown using `[video name](./file.mp4)`. Some viewers may also support embedded HTML video, but links are the safest option.
